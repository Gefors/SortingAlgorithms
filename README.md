#     SortingAlgorithms

A Java application that uses the sorting algorithms:

1. Insertion Sort (InsertionSorter.java)
2. Merge Sort (MergeSort.java)
3. Modified Merge Sort (ModifiedMergeSort.java)
4. Quick Sort (QuickSort.java)
5. Modified Quick Sort (ModifiedQuickSort.java)
6. TreeMap-based Sort (TreeMapSort.java)
